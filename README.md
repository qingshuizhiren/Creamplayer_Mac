# Creamplayer_Mac

一个基于Electron的音乐播放器应用，根据[Creamplayer](https://github.com/Beadd/Creamplayer)修改而来。

## 开发环境设置

1. 安装依赖：
```bash
npm install
```

2. 运行开发模式：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 打包应用：
```bash
# macOS
npm run make:mac
```
